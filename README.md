
# Starter Kit

This is a starter for Vue inside laravel with vue-router , Vuex , Sass.

## Installation

Install my-project with npm

```bash
  Git Clone https://github.com/peyas4854/VueSpa-Laravel
  cd repo_folder 
```

## Install Dependencies


```bash
  composer i
  npm i
```


## Run Locally



```bash
  php artisan serve
```
